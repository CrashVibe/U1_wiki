# 成分姬

看看他的成分吧，我去，重度二刺螈？我超，是嘉然？？

## 服务信息

- **服务名**：成分姬
- **说明**：查询 B 站用户关注的 VTuber 成分

## 命令列表

### 查成分 `<UID | B站名>`

查询某个 B 站用户关注了哪些 VTuber，分析出 TA 的「成分」，并渲染成一张图片返回。

- 参数可以填 **UID**（纯数字）或 **B 站用户名**（会自动搜索匹配第一个用户）
- 例如：
  - `查成分 413417479`
  - `查成分 Mrling-XD`

  （上面两个都是作者的 B 站号（

## 注

- VTuber 名单每 3 小时自动更新一次，保证成分识别尽量准确。
