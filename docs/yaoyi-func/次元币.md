# 次元币

赚取次元币，开启你的二次元交易之旅吧！次元币是 U1 的通用货币，多个功能都会用到它。

## 服务信息

- **服务名**：次元币
- **说明**：查看你的次元币余额

## 命令列表

### 次元币

查看你当前的次元币余额。

## 怎么赚 / 怎么花

| 操作 | 次元币 |
| --- | --- |
| [钓鱼](./钓鱼) 卖鱼 | 💰 赚取（越长越高品质越值钱） |
| [娶群友](./娶群友) 娶第 3 个及以后的对象 | 💸 支付聘礼 |
| [娶群友](./娶群友) 冷静期内离婚 | 💸 支付罚款 |

> 余额不足时，需要花钱的操作会失败，所以平时多钓鱼攒点家底吧~
