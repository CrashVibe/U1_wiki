# 钓鱼

赛博钓鱼，值得你拥有！甩一杆，运气好能钓到隐火神鱼，运气差……就钓到一条「996福报鱼」。

## 服务信息

- **服务名**：钓鱼
- **说明**：赛博钓鱼，攒鱼竿、卷排行榜、卖鱼换次元币

## 命令列表

### 钓鱼

> 别名：`fishing`

甩杆开钓！发送后机器人会先给你的消息贴上 🐳 表情，等待 1~6 秒后揭晓结果，钓到鱼会换成 🎉。

- **冷却时间**：默认 `15` 秒，冷却中再发会被无视
- 每钓一次，鱼竿经验 +1，达到要求会自动升级（详见 [鱼竿系统](#鱼竿系统)）
- 钓到的鱼会自动放进背包，每条鱼都有自己的**品质**和**长度**

> 私聊钓鱼时，机器人会发送一张社区画师手绘的「姚奕在钓鱼」插画代替表情反馈ww

### 背包

查看背包里所有的鱼，按品质分组展示。私聊 / QQ 官方机器人下为纯文本，其他平台用合并转发。

### 统计信息

查看你的钓鱼档案：

- 总钓鱼次数
- 背包中所有鱼的总长度（cm）
- 当前鱼竿等级与升级进度
- 连续倒霉次数（连续钓到低品质鱼的计数）

### 鱼竿

> 别名：`我的鱼竿`

查看你的鱼竿详情：当前等级、升级进度（还差几次升级）、各品质的权重加成、特殊鱼加成。

### 卖鱼 `<鱼名 | 全部 | 品质>`

学林北卖鱼，走十年弯路~ 三种卖法：

- **卖鱼 `全部`**：清空背包，一次性全卖
- **卖鱼 `<品质>`**：卖掉某一品质的所有鱼，品质填 `腐烂`/`发霉`/`普通`/`金`/`虚空`/`隐火`
- **卖鱼 `<鱼名>`**：卖掉指定名字的一条鱼

> 售价公式：**鱼的长度(cm) × 该品质单价**。所以又长又高品质的鱼最值钱。

### 钓鱼开关

钓鱼功能的群级开关，_默认开启_。

- 仅 **管理员 / 群主** 可用
- QQ 官方机器人暂不支持此功能

### 钓鱼排行榜

> 别名：`fish_leaderboard`

查看**本群**的钓鱼榜单，每个榜单只显示前 3 名。仅当天有对应数据时才会显示该榜单。共五种：

| 榜单 | 含义 |
| --- | --- |
| 钓鱼次数榜 | 钓得最勤的人 |
| 欧皇榜 | 钓到高品质鱼最多的人 |
| 非酋榜 | 钓到低品质鱼最多的人 |
| 连续倒霉榜 | 连续钓到低品质鱼的纪录 |
| 经验榜 | 鱼竿经验最高的人 |

## 鱼的品质

鱼共有 6 种品质，越稀有越值钱、个头也越大。基础出现权重越低越难钓：

| 品质 | 稀有度 | 长度范围 | 单价(每 cm) |
| --- | --- | --- | --- |
| 腐烂 | 常见（倒霉） | 15~45 cm | 0.05 |
| 发霉 | 常见（倒霉） | 20~150 cm | 0.08 |
| 普通 | 最常见 | 1~30 cm | 0.10 |
| 金 | 稀有 | 125~800 cm | 0.15 |
| 虚空 | 史诗 | 800~4000 cm | 0.20 |
| 隐火 | 传说 | 1000~4000 cm | 0.20 |

每种品质下都有几十条花样百出的鱼，部分鱼还有专属吐槽文案，钓到时一起发给你（比如「咸鱼本鱼」「珊瑚宫心海」「MrlingXD」？！）。

::: tip 运势加成
如果开启了 [今日运势](./今日运势)，你当天的**幸运星**会影响钓鱼概率——星越多，金 / 虚空 / 隐火等高品质鱼的概率越高，腐烂 / 发霉的概率越低。出门前先看看运势准没错。
:::

## 鱼竿系统

鱼竿共 5 级，从低到高：**普通 → 银 → 金 → 圣物 → 虚空**（最高级）。等级越高，越容易钓到高品质鱼。

### 升级

每钓一次累计经验，达到当前等级的「升级所需次数」就自动升级：

| 当前等级 | 升到下一级所需钓鱼次数 |
| --- | --- |
| 普通 → 银 | 50 |
| 银 → 金 | 150 |
| 金 → 圣物 | 300 |
| 圣物 → 虚空 | 500 |
| 虚空 | 已满级 |

### 降级

鱼竿也会「掉级」，触发条件：

- **钓到倒霉鱼**：钓到特定的「倒霉鱼」（如 996福报鱼、破产倒闭鱼等）时，有小概率降级
- **连续倒霉**：连续钓到 `8` 条低品质鱼（腐烂 / 发霉）后，降级概率大幅提升
- **长期不钓**：超过 `7` 天没钓鱼，鱼竿会「生锈」直接降级

> 等级越高的鱼竿，降级基础概率越低，越扛造。
