# 娶群友

把群友统统娶回家，结芬，一胎生八个！

支持一对多——你可以同时拥有多个对象，还能用 `婚姻图谱` 看看全群的连理关系。

## 服务信息

- **服务名**：娶群友
- **说明**：每天找群友增进感情，查看本群的婚姻关系图谱

## 命令列表

### 娶群友 `[@群友]`

随机娶一个群友，或者 `@` 指定一个群友，成为你的对象。

- **随机**：有 `90%` 的几率成功娶到一个群友，否则今天一整天单身（哈哈
- **指定（@对象）**：有 `40%` 的几率成功娶到 TA，失败时会随机帮你再分配一个 CP
- 不能娶自己、不能娶机器人，也不能重复迎娶同一个人

::: tip 一对多与聘礼
你可以同时拥有多个对象。前 **2** 个对象免费，从第 3 个开始需要支付次元币聘礼，且**指数增长**：

> 聘礼 = `50 × 2^(已有对象数 − 2)`

也就是第 3 个 `50`、第 4 个 `100`、第 5 个 `200`……钱包不够就只能忍痛放弃啦。
:::

### 离婚 `[@对象]`

和你的某个对象离婚。如果你同时有多个对象，需要 `@` 指定要和谁离婚。

- 每段婚姻有 **1 小时** 的离婚冷静期（从娶到 TA 的那一刻算起）
- **冷静期已过**：免费离婚
- **冷静期内**：需要支付罚款才能离，罚款同样**指数增长**：

> 罚款 = `200 × 2^(今日涉及的关系数)`

钱包不够就离不了，老老实实等冷静期吧～

### 婚姻图谱

> 别名：`关系图谱`

把本群当前所有在婚关系渲染成一张关系图，谁和谁连在一起一目了然。

- 自动按连通关系分组，环形排布，头像 + 昵称展示
- 如果群里还没有人结婚，会提示你先去 `娶群友` 试试

## 注

每天凌晨 `0` 点会重置娶群友的数据，重置后可以重新娶群友，离婚冷静期也会一并清空。
