# 排行榜奖励

每周和每月自动结算钓鱼与早晚安排行榜，给排名靠前的用户发放次元币奖励。

## 服务信息

- **服务名**：排行榜奖励
- **说明**：定期结算排行榜并发放次元币奖励

## 奖励规则

### 每周奖励（每周一 0:00 结算）

| 排名    | 钓鱼次数榜 | 早晚安早安排行榜 |
| ------- | ---------- | ---------------- |
| 第 1 名 | 500 次元币 | 300 次元币       |
| 第 2 名 | 300 次元币 | 200 次元币       |
| 第 3 名 | 150 次元币 | 100 次元币       |

### 每月奖励（每月 1 日 0:00 结算）

| 排名    | 钓鱼次数榜  | 早晚安早安排行榜 |
| ------- | ----------- | ---------------- |
| 第 1 名 | 1500 次元币 | 800 次元币       |
| 第 2 名 | 800 次元币  | 500 次元币       |
| 第 3 名 | 400 次元币  | 300 次元币       |

> 获奖者会收到私信通知，奖励直接发放到次元币余额。
