## 服务信息

- **服务名**：今日运势
- **说明**：抽一支今日签，看看运势星级（tips：跟钓鱼联动喔！）

## 命令列表

### 运势

> 别名：`今日运势`

抽取你今天的签文，每人每天一支。返回内容包含四部分：

- **运势**：从「凶」到「大吉」，大吉还可能附带 `官运` / `财运` / `才艺` 等加成标签
- **星级**：`★` 数量，从 0 星到 7 星，星越多今天越旺
- **签文**：一句古风签文
- **解签**：白话解读

::: tip 每天刷新
运势每天刷新一次。同一天内反复发「运势」拿到的都是同一支签，第二天才会重新抽。
:::

## 运势等级

签库共 77 支签，运势由差到好大致分布如下：

| 运势                  | 含义               |
| --------------------- | ------------------ |
| 凶                    | 最差，诸事不顺     |
| 凶带吉 / 吉带凶       | 喜忧参半           |
| 中吉                  | 还不错             |
| 吉 / 大吉             | 顺风顺水           |
| 大吉 + 官运/财运/才艺 | 顶配，附带额外加成 |

星级与运势对应，最低 `☆☆☆☆☆☆☆`（0 星），最高 `★★★★★★★`（7 星）。

## 跟钓鱼联动

你当天抽到的**星级**会直接影响 [钓鱼](./钓鱼) 的出鱼概率：星越多，金 / 虚空 / 隐火等高品质鱼越容易上钩，腐烂 / 发霉的概率则会降低。所以——**先抽运势，再去钓鱼**，欧皇之路从这里开始。

## 签到奖励

每次查看运势时自动完成签到，获得次元币奖励：

- **基础奖励**：运势星级 × 15 次元币（1★=15, 7★=105）
- **连续加成**：连续签到天数 × 5 次元币（连续 3 天 +15，上限连续 30 天 +150）
- 当日重复查看不会重复发币

> 签到信息会附加在运势签文的末尾，方便查看今日收益和连续天数。
